# **App Name**: Ai skin disease detection

## Core Features:

- Image Upload: Allow users to upload images of skin areas for analysis via drag and drop or browse.
- AI-Powered Screening: Analyze uploaded images using AI to identify potential skin conditions and generate a confidence score for each condition.
- Instant Preview: Provide users with an instant preview of the uploaded image within a clean, medical-tech UI.
- Confidence-Based Results: Display top condition predictions with confidence bars, labels, and concise explanations.
- Safety Prompts: Provide guidance notes and safety prompts, ensuring users understand that the tool is not a diagnosis.
- Image analysis with a reasoning tool: The tool uses LLM reasoning to generate image analysis; it uses information from a skin condition database and uses skin condition images for comparison in deciding the final confidence interval

## Style Guidelines:

- Primary color: Deep blue (#1D4ED8), conveying trust and reliability, aligning with the medical-tech theme.
- Background color: Light blue (#F7FAFF), providing a clean and professional feel.
- Accent color: Sky blue (#0EA5E9), used for interactive elements and highlights, ensuring clarity and focus.
- Body and headline font: 'Inter' sans-serif, offering a modern, clean, and easily readable experience.
- Use clear, simple icons with a touch of blue to represent different features and actions, ensuring easy understanding.
- Subtle transitions and animations when loading images and displaying results to enhance user experience without being distracting.
